# Flutter Auth Todo App

A clean Todo App built with Flutter with Signup/Login and local storage.

## Features
- User Signup & Login
- SharedPreferences storage
- Task management
- Category view
- APK build supported

## Tech Stack
Flutter
Dart
SharedPreferences
